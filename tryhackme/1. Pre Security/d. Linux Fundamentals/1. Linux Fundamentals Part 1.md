4. Navigasi & Manipulasi File

ls â†’ daftar file & folder.

5. Mencari File & Isi

6. Shell Operators

& â†’ jalankan perintah di background.

&& â†’ jalankan perintah kedua jika perintah pertama sukses.

> â†’ redirect output, menggantikan isi file.

>> â†’ redirect output, menambahkan ke akhir file  .


Contoh:

echo password123 > passwords
echo tryhackme >> passwords

7. Kesimpulan

Perintah dasar Linux: navigasi (cd, ls, pwd), tampil teks (echo, cat), cari (find, grep).

Shell operators mempermudah automasi dan kontrol.

Semua komando ini adalah dasar yang wajib dikuasai untuk penggunaan Linux.
