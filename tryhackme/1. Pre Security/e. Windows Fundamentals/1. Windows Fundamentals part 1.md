## 🖥️ Windows Fundamentals Part 1 – Ringkasan Lengkap

### 📌 Task 1 – Pendahuluan

* Pelajari dasar Windows OS seperti desktop, file system NTFS, UAC, Control Panel, Task Manager, dll. Mesin virtual akan disediakan. ([TryHackMe][1])

---

### 🪟 Task 2 – Windows Editions

* Windows edisi **Pro mendukung BitLocker**, sedangkan edisi Home tidak. ([Medium][2])

---

### 🖱️ Task 3 – Desktop & GUI

* Setting untuk **menyembunyikan Search Box**: pilih opsi `Hidden`
* Setting untuk **menyembunyikan Task View button**: pilih `Show Task View button`
* Ikon lain di area notifikasi selain Clock dan Network: **Action Center** ([Medium][3])

---

### 🗂️ Task 4 – File System

* **NTFS** = *New Technology File System*, file system modern Windows dengan fitur seperti enkripsi, kompresi, dan permission granular ([Medium][3])

---

### 📁 Task 5 – Folder Windows\System32

* Variabel sistem untuk folder Windows: **`%windir%`** (biasanya menunjuk ke `C:\Windows`) ([Medium][3])

---

### 👥 Task 6 – Akun Pengguna & Permissions

* Nama akun pengguna lain: **`tryhackmebilly`**
* Grup yang diikuti: **Remote Desktop Users, Users**
* Akun bawaan untuk guest: **Guest** (dalam keadaan *disabled*) ([Medium][3])

---

### 🔐 Task 7 – User Account Control (UAC)

* **UAC** = *User Account Control*; fitur keamanan yang meminta konfirmasi ketika program membutuhkan hak istimewa tinggi ([Medium][3], [Medium][2])

---

### ⚙️ Task 8 – Control Panel

* Saat melihat Control Panel dengan tampilan “Small icons”, pengaturan terakhir yang muncul: **Windows Defender Firewall** ([Medium][3])

---

### 📊 Task 9 – Task Manager

* Shortcut membuka Task Manager: **Ctrl + Shift + Esc** ([Medium][3])
