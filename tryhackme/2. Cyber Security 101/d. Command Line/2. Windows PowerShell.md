## **Tugas 1 & 2: Introduction & What is PowerShell?**

**Introduction (Perkenalan)**
PowerShell adalah shell baris perintah dan bahasa scripting yang dikembangkan oleh Microsoft untuk administrasi sistem dan otomatisasi tugas.
PowerShell menggabungkan kemampuan **Command Prompt** dengan bahasa pemrograman berbasis objek .NET.

**What is PowerShell? (Apa itu PowerShell?)**

* PowerShell dapat menjalankan **cmdlet** (perintah kecil untuk tugas spesifik), skrip, dan program eksternal.
* Berbasis objek: output dari cmdlet adalah objek, bukan teks mentah.
* Dapat digunakan untuk mengelola sistem operasi, file, jaringan, dan layanan secara lokal maupun jarak jauh.

## **Tugas 4: Navigating the File System and Working with Files (menavigasi sistem file dan bekerja denangan file)**

* **`Get-ChildItem -Path ".\Documents"`** â†’ Menampilkan isi folder Documents (`ls` jika di Linux).
* **`Set-Location -Path ".\Documents"`** â†’ Berpindah ke folder Documents (`cd` jika di Linux).
* **`New-Item -Path ".\Documents" -ItemType "Directory"`** â†’ Membuat folder baru.
* **`Remove-Item -Path ".\Documents"`** â†’ Menghapus folder atau file.
* **`Copy-Item -Path ".\Documents\text.txt" -Destination ".\Documents\Contoh\text2.txt"`** â†’ Menyalin file.
* **`Get-Content -Path ".\Documents\text.txt"`** â†’ Membaca isi file text.txt (`cat` jika di Linux).

## **Tugas 5: Piping, Filtering, and Sorting Data (perpipaan, penyaringan, dan pengurutan data)**

* Cmdlet mengembalikan **objek** â†’ Objek dapat diproses lagi menggunakan `|` cmdlet.
  **Contoh:**

```powershell
Get-ChildItem | Where-Object -Property "Extension" -eq ".txt"
```

**Operator Perbandingan di PowerShell:**

* **`-ne`** â†’ Tidak sama dengan.
* **`-gt`** â†’ Lebih besar dari (strict).
* **`-ge`** â†’ Lebih besar dari atau sama dengan (non-strict).
* **`-lt`** â†’ Kurang dari (strict).
* **`-le`** â†’ Kurang dari atau sama dengan (non-strict).

**Cmdlet Filtering & Selecting:**

* **`Where-Object`** â†’ Memfilter objek dengan kondisi tertentu.

  ```powershell
  Get-ChildItem | Where-Object -Property "Length" -gt 100
  ```
* **`Select-Object`** â†’ Memilih properti tertentu atau membatasi jumlah objek.

  ```powershell
  Get-ChildItem | Select-Object Name, Length
  ```
* **`Select-String`** â†’ Mencari teks di dalam file (mirip `grep` di Unix).

  ```powershell
  Select-String -Path ".\captain-hat.txt" -Pattern "hat"
  ```
